# Pythonification

Python learning repo focused on becoming productive quickly as a software engineer, not a data scientist.

## Approach

This repo starts simple on purpose.

For the first phase, this should mostly be:

- small scripts
- focused exercises
- refactoring passes
- short notes in Markdown

It should not start as a full multi-module project. That would add structure before the Python basics feel natural.

## Why Start with Scripts

Right now the main goal is to:

- read Python comfortably
- write idiomatic transformations
- understand core built-in types
- get used to Python syntax and standard patterns

A package-heavy structure would distract from that.

Once the basics are stable, this repo can evolve into a more realistic application layout with:

- modules
- tests
- `venv`
- `pyproject.toml`
- linting and formatting

That transition is planned for Week 4 and later.
