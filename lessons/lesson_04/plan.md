# Lesson 04 Plan

## Theme

Week 1 consolidation.

This lesson mixes the earlier concepts together so you stop treating them as separate syntax topics.

## Objectives

By the end of this lesson, you should be able to:

- read a small script and quickly identify the right built-in collection type
- combine filtering, counting, mapping, and simple record processing
- write a short script that feels natural in Python
- notice where your code is becoming too Java-shaped and simplify it

## Time Budget

Target total: about 60-75 minutes.

Suggested split:

- 10-15 minutes scanning the lecture
- 35-45 minutes solving the exercise
- 15 minutes reviewing and refactoring

## Workflow

1. Read [lecture.md](/Users/michael.linka/Udemy/pythonification/lessons/lesson_04/lecture.md)
2. Work in [exercise.py](/Users/michael.linka/Udemy/pythonification/lessons/lesson_04/exercise.py)
3. Aim for a complete working script first
4. Then tighten the code where Python offers a cleaner expression
5. Bring the result for review

## What Success Looks Like

Success means:

- you can solve the exercise without freezing on syntax
- you can explain why you chose `list`, `dict`, or `set`
- your final code is short, readable, and not over-abstracted

## Common Mistakes to Avoid

- introducing helper functions too early for tiny tasks
- stacking multiple transformations into one unreadable expression
- optimizing before the script is correct
- treating review as optional instead of part of the lesson
